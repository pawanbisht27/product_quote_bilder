# Product Quote Builder

A clean and minimal Flutter app for generating professional business quotes with real-time preview and calculations.


## Overview

*Product Quote Builder* is a modern Flutter application designed to help users easily create, preview, and share product quotations with clients.  
The app focuses on simplicity, accuracy, and user-friendly design — built entirely using Flutter’s frontend capabilities.


## Features

- Add client details including name, address, and reference.
- Add multiple product items with quantity, price, and tax.
- Automatic calculation of subtotal, tax, and grand total.
- Real-time quote preview screen before sharing.
- Included screenshots for visual reference.
- Clean and responsive UI built with Flutter Material components.


## Tech Stack

- *Framework:* Flutter
- *Language:* Dart
- *Architecture:* Stateless and Stateful Widgets
- *UI Toolkit:* Material Design
